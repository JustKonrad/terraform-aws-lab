# Terraform AWS Lab 🚀

Beginner Terraform project deploying AWS infrastructure.

## Technologies
- Terraform
- AWS
- S3
- Infrastructure as Code (IaC)

## Features
- Creates an AWS S3 bucket
- Enables bucket versioning
- Uses Terraform provider configuration

## Project Structure
```bash
.
├── main.tf
├── provider.tf
├── variables.tf
├── outputs.tf
├── ec2.tf
├── .gitignore
└── README.md
```

## Usage

Initialize Terraform:
```bash
terraform init
```

Preview changes:
```bash
terraform plan
```

Deploy infrastructure:
```bash
terraform apply
```
